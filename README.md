# Cellular-Automated-External-Defibrillator
A **cellular automated external defibrillator (AED)** that detects sudden cardiac arrest, delivers life-saving shocks, and uses cellular connectivity to automatically alert emergency services with real-time location and event data.The device is designed for reliability, rapid response, and ease of use, making it suitable for both trained professionals and public emergency scenarios.

🏗️ **System Architecture**

The CAED system is divided into two primary modules:

High Voltage Board (HV-B) – Handles energy generation, storage, and delivery

Control Board (C-B) – Manages logic, user interaction, and system monitoring
